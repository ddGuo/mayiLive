# 直播原理学习研究
准备写个直播助手工具，先挖个坑

最终目标是根据弹幕，礼物，点赞，关注，等操作去触发各种好玩的东西
比如 触发视频，语音，按键，物联网设备 等等

# 环境大于3.10应该都可以

```
创建环境
conda create --name your_env_name python=3.10
激活环境
source activate your_env_name
pip按照需要的包 
pip install -r requirement.txt

运行项目
python main.py
```
